# PWD project

### How to build and run me

Clone our git repository:

```
git clone https://github.com/pwd-project/pwd.git
cd pwd
```

Build project and fire tests:

```
gradlew test
```

Run PWD service at `http://localhost:8080`

```
gradlew run
```

### Project version
Based on GIT tags. 
Check the `currentVersion` number

```
gradlew cV
```
